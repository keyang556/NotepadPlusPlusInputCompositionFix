Notepad++ 組字區修正

這個 NVDA 附加元件修正 64 位元 Notepad++ 8.3 以上版本中的輸入法組字區朗讀問題。

它處理 nvaccess/nvda#14152：使用 NVDA 2022.1 以後版本搭配 Notepad++ 時，在中文輸入法尚未送出的組字字串內移動游標，可能只聽到空白，點字顯示器也不會更新。

附加元件會保留 NVDA 內建的 Notepad++ 64 位元 Scintilla TextInfo 支援；只有在作用中的輸入法組字或讀音字串存在時，才優先使用 NVDA 的輸入法組字 TextInfo，讓游標移動能像其他應用程式一樣正常讀出組字區內容。

相容性提醒

這個附加元件會覆寫 NVDA 的 Notepad++ app module，可能會和其他同樣自訂 Notepad++ 的 NVDA 附加元件衝突。安裝本附加元件前，請先解除安裝或停用其他 Notepad++ 相關附加元件。
